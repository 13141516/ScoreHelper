#配置
1.数据库的配置：数据库的配置文件在c3p0.xml中，数据库表名为customer,表结构请参考<br />
![image](https://github.com/13141516/ScoreHelper/raw/master/CarPro/screenshot/three.png)<br />
2.邮箱发送账号：邮箱发送账号和密码的设置在score.py文件中<br />
